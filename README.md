# 🌍 Global Startup Funding Analysis Dashboard

## 📊 Overview

This project presents a global startup funding analysis dashboard built using Power BI.

The dataset used in this project is a **sample/generated dataset (1000 rows)** created for learning and visualization purposes.

--

## 🚀 Features

* KPI metrics (Total Funding, Total Startups)
* Funding trend over time
* Country-wise funding analysis
* Industry-wise funding distribution
* Interactive filters (Country, Industry, Year)
* Geospatial visualization using map
